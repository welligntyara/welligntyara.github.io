Seção 2 — Ranking de Tecnologias

  1. HTML

    HTML é a base da construção de páginas web.

  2. CSS

    CSS é utilizado para estilizar páginas e melhorar a aparência visual.

  3. JavaScript

    JavaScript adiciona interatividade aos sites.

  4. React

    React facilita a criação de interfaces modernas.

  5. Node.js

    Node.js permite utilizar JavaScript no backend.

Seção 3 — Glossário Técnico

HTML
Linguagem de marcação utilizada para estruturar páginas web.
CSS
Linguagem responsável pela estilização das páginas.
JavaScript
Linguagem de programação utilizada na web.
Pode ser utilizada tanto no frontend quanto no backend.
DOM
Estrutura que representa os elementos da página.
API
Interface de comunicação entre sistemas.
Permite integração entre aplicações.
URL
Endereço Web
Localização de um recurso na internet.
HTTP
Protocolo Web
Protocolo utilizado para transferência de dados na web.
DNS
Sistema responsável por converter nomes de sites em endereços IP.
